# nasa-federal-spending-analysis
PostgreSQL (Star Schema) ve Python ile NASA FY24 harcamalarını analiz eden uçtan uca veri projesi. İleri seviye SQL teknikleriyle pazar yoğunlaşması ve bütçe dinamikleri hesaplanmış, Plotly ile interaktif olarak görselleştirilmiştir. Finansal verileri stratejik içgörülere dönüştüren profesyonel bir portfolyo çalışmasıdır.
